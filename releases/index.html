Development Builds

Preview what's coming next for Pinta by downloading a development build. These builds are updated periodically, giving you early access to new features and improvements.

However, development builds are primarily intended for testing purposes and may contain bugs or unfinished features.

As a result, they are not recommended for production work. If you'd like to test the latest changes, you can also download builds from each revision (requires being logged in to a GitHub account).
